# Python Telegram Bot Template

## Instalacion
1. Crear un entorno virtual con
```
python -m venv NOMBRE_VENV
``` 

2. Activar entorno
```
source ./NOMBRE_VENV
```

3. Instalar requerimientos
```
pip install -r ./requirements.txt
```

4. Configurar variables de entorno
```
python crear_env.py
```

## Ejecucion

```
python main.py
```
